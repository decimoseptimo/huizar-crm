Huizar-POS
============================

A basic web front-end for a point-of-sale desktop application.
#### [Live demo](http://www.azulacero.mx/demos/huizar-pos/web/)

Requirements
------------
- Yii 2.0.8
- MySQL

Instructions
------------
- Setup database
- Setup /config/db.php file with your db credentials
